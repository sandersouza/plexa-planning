# Milestones

Espelho local do backlog remoto organizado por milestone.

Cada pasta deve conter:
- `README.md` com o identificador da milestone remota
- `CHALLENGES.md` com desafios, decisões e encaminhamentos
- um arquivo por issue no formato `<numero>-<slug>.md`

Sincronize este diretório sempre que o backlog remoto mudar de forma relevante.
